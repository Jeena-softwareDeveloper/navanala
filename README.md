# Navanala Software Private Limited Website

This is a modern, animated website for Navanala Software Private Limited, built with React, Tailwind CSS, and Framer Motion.

## Features
- **Modern UI**: Glassmorphism, gradients, and sleek dark mode.
- **Animations**: Entrance animations and hover effects using Framer Motion.
- **Responsive**: Fully responsive design for mobile and desktop.
- **Products**: Sections for ERP, CRM, and other SaaS solutions.

## Setup

1.  **Install dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Lucide React
- Vite
